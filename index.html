Bienvenue sur le Serveur Minecraft !

Suivez les étapes ci-dessous pour vous connecter.

1. Adresse du Serveur

Pour vous connecter, vous aurez besoin de l'adresse IP du serveur :

#Je vous la donne (ou l'adresse IP publique)

Port : 19132

2. Connexion sur PC (Windows 10/11)

  1. Ouvrez Minecraft Bedrock sur votre PC.
  2. Allez dans l'onglet "Play" (Jouer).
  3. Cliquez sur l'onglet "Servers" (Serveurs) en haut.
  4. Faites défiler vers le bas et cliquez sur "Add Server" (Ajouter un serveur).
  5. Remplissez les champs :
    • Server Name : Donnez un nom (ex: "Mon Serveur").
    • Server Address : Entrez l'adresse IP du serveur.
    • Port : 19132.
  6. Cliquez sur "Save" (Sauvegarder).
  7. Sélectionnez votre serveur dans la liste et cliquez sur "Join Server" (Rejoindre le serveur).

4. Configurer le DNS sur les Consoles

Sur les consoles (PlayStation, Xbox, Nintendo Switch), vous devez configurer un DNS personnalisé pour accéder au serveur.

PlayStation (PS4/PS5)

  1. Allez dans Paramètres > Réseau > Configurer la connexion Internet.
  2. Sélectionnez votre connexion Wi-Fi ou filaire.
  3. Choisissez Personnaliser pour les paramètres réseau.
  4. Lorsque vous arrivez à l'étape DNS, sélectionnez Manuel.
  5. Entrez l'adresse DNS suivante :
    • DNS Primaire : 104.238.130.180
    • DNS Secondaire : 1.1.1.1 (ou laissez vide).
  6. Terminez la configuration et testez la connexion.

Besoin d'aide visuelle ? Regardez cette vidéo : Tutoriel PS4/PS5.

Xbox (One/Series X|S)

  1. Allez dans Paramètres > Réseau > Paramètres réseau.
  2. Sélectionnez Paramètres avancés > DNS.
  3. Choisissez Manuel.
  4. Entrez l'adresse DNS suivante :
    • DNS Primaire : 104.238.130.180
    • DNS Secondaire : 1.1.1.1 (ou laissez vide).
  5. Enregistrez les modifications et testez la connexion.

Besoin d'aide visuelle ? Regardez cette vidéo : Tutoriel Xbox.

Nintendo Switch

  1. Allez dans Paramètres > Internet > Paramètres Internet.
  2. Sélectionnez votre connexion Wi-Fi.
  3. Choisissez Modifier les paramètres.
  4. Allez dans DNS et passez en mode Manuel.
  5. Entrez l'adresse DNS suivante :
    • DNS Primaire : 104.238.130.180

5. Problèmes de Connexion ?

Si vous ne parvenez pas à vous connecter :

  • Vérifiez que l'adresse IP et le port sont corrects.
  • Assurez-vous que le serveur est en ligne.
  • Contactez l'administrateur du serveur pour plus d'aide.
